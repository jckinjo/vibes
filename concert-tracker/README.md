# Concert Tracker

Live concert schedule aggregator for acoustic guitarists

## Supported Artists

- GOGO SATOSHI
- DEPAPEPE
- Kotaro Oshio (押尾コータロー)
- Seiji Igusa (井草聖二)
- Tatsuya Maruyama (丸山達也)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3456

API: `GET /api/concerts` (use `?refresh=1` to force refresh)
